Features

Everything your homelab display needs

Built specifically for touchscreen panels - drag-and-drop layout, live data, and a beautiful dark UI.

⚡
Real-time data
Server-sent events push widget updates live. No page refresh needed - data flows in as it changes.
👆
Touchscreen-first
Designed for always-on panel displays. Scroll to flip stacked widgets, tap to interact - no mouse required.
🏠
Fully self-hosted
Runs on a single Python process with SQLite. No Docker required, no cloud account, no telemetry.
📐
Drag-and-drop editor
Visual layout editor lets you position, resize, and stack widgets on a pixel-perfect grid. Changes save instantly.
🗂️
Multi-page & stacks
Organize widgets across multiple pages. Stack single-page widgets in one card and scroll between them.
🎨
Beautiful themes
Seven color palettes, multiple card styles, custom gradients, glow effects, and accent colors - fully configurable.
🔐
Encrypted backups
Export your entire config - layout, integrations, settings - as an encrypted backup. Restore on any instance.
👥
Multi-user
Add multiple users with their own credentials. The display page is read-only, perfect for shared panels.
🔌
Easy integrations
Connect services via a guided wizard. API credentials are encrypted at rest with Fernet symmetric encryption.

Get started

Up and running in minutes

No Docker, no cloud account. Just Python and a git clone.

1
Clone the repo
Grab the latest code from GitHub.
git clone https://github.com/parkerschoenig/TouchBoard.git cd TouchBoard
2
Start the server
Python 3.10+ required. The script creates a virtualenv, installs dependencies, and starts the server on port 8011.
bash scripts/run.sh
3
Open the editor & add widgets
Default credentials are admin / admin@touchboard - you'll be prompted to change them on first login.
http://<your-server-ip>:8011
4
Point your display
Open the display URL on your touchscreen panel in full-screen kiosk mode.
http://<your-server-ip>:8011/display
