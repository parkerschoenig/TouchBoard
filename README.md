# TouchBoard — Live Demo

This is the demo instance of TouchBoard. Data is simulated — no real services are connected.

**Demo credentials:** `admin` / `touchboard`

> The demo is read-only. Edits to the layout and settings are blocked.

---

## What is TouchBoard?

A touchscreen-first monitoring dashboard for homelabs and self-hosted infrastructure. Design a multi-page layout of widgets, integrations, and stacks in any browser — then point any display at the server URL.

## Widgets & Integrations

| Type                | Notes |
|---------------------|-------|
| **Clock**           | Digital or analog, 12h/24h, any IANA timezone |
| **Weather**         | Current conditions + forecast via Open-Meteo |
| **Ping**            | HTTP, TCP, or ICMP host availability |
| **Proxmox**         | CPU, memory, uptime, VM/CT summary |
| **TrueNAS**         | Pool storage, memory, and CPU |
| **NetBox**          | Rack elevation, IPs, and devices |
| **OPNsense**        | Firewall traffic and system health |
| **AdGuard Home**    | DNS query stats and block rate |
| **Google Calendar** | Upcoming events from Google Calendar or any ICS feed |
| **Stream**          | Live camera feed via HLS or RTSP |
